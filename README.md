# React Weather App

A web application built using React that displays real-time weather information for different cities.

## Tech Stack
- React
- JavaScript (ES6)
- CSS
- Weather API

## Features
- Search weather by city name
- Displays temperature and weather conditions
- Responsive user interface

## What I Learned
- API integration in React
- State and component management
- Handling user input
